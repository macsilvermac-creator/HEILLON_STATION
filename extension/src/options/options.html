Heillon Legal — Configuração

A extensão captura interações com IA jurídica e envia para o substrato Heillon. Configure abaixo.

1. Conexão

Use http://127.0.0.1:8000 para desenvolvimento local.

Gere em Console → Conta → Chaves de API.

2. Provedores de IA monitorizados

Desmarque os provedores em que NÃO quer que a extensão capture interações (opt-out).

Mudanças são aplicadas imediatamente — sem necessidade de recarregar a página.

3. Privacidade

  • A chave de API fica armazenada APENAS neste dispositivo (chrome.storage.local).
  • Prompts e respostas são enviados apenas para o servidor configurado.
  • Cada captura gera 1 HDR com selo ICP-Brasil — consulte em Console → Conta.
  • Para parar imediatamente: desmarque todos os provedores acima ou apague a chave.
