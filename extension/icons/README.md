# Icons

Required for Chrome Web Store submission:
- `icon-16.png` — 16×16, used in extension menu
- `icon-48.png` — 48×48, used in extensions page
- `icon-128.png` — 128×128, used in Chrome Web Store listing

**Design brief (when commissioning real icons):**
- Background: solid dark `#0A0E1A` (Heillon deep space)
- Foreground: stylized "H" or chain-of-custody motif in gold `#D4AF37`
- Avoid text/letterforms below 16px (illegible)
- Provide also `icon-512.png` (square promo) for Chrome Web Store listing
- Optionally provide promotional tile (440×280) and large tile (920×680) for store listing
